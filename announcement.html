No Many Friends

Bot Discord ini dirancang untuk membantu Anda mengelola server dengan lebih baik, memberikan perintah yang berguna, dan meningkatkan pengalaman komunitas Anda.

Announcement

Fitur pengumuman memungkinkan Anda untuk membuat, mengedit, dan menghapus pengumuman di server Discord Anda. Dengan perintah ini, Anda dapat dengan mudah mengelola informasi penting yang perlu disampaikan kepada anggota server.

/announcement-create

Perintah ini digunakan untuk membuat pengumuman di saluran Discord. Berikut langkah-langkahnya:

  1. Ketik /announcement-create untuk memulai.
  2. Pilih jenis pengumuman: Text untuk teks biasa atau Embed untuk format embed.
  3. Pilih saluran teks yang ingin Anda gunakan untuk mengirim pengumuman.
  4. Jika memilih Text, masukkan teks pengumuman Anda. Ketik cancel untuk membatalkan.
  5. Jika memilih Embed, ikuti panduan untuk menambahkan judul, deskripsi, dan footer sesuai kebutuhan. Ketik cancel untuk membatalkan.
  6. Setelah semua informasi dimasukkan, pengumuman akan dikirim ke saluran yang dipilih.

Gunakan perintah ini untuk menyampaikan informasi penting kepada anggota server Anda dengan mudah.

/announcement-edit

Perintah ini digunakan untuk memperbarui pengumuman yang sudah ada. Untuk menggunakan perintah ini, Anda perlu:

  1. Ketik /announcement-update [id pengumuman] [pesan] untuk memperbarui pengumuman.
  2. Pastikan Anda memiliki izin yang diperlukan untuk menggunakan perintah ini.

/announcement-delete

Perintah ini digunakan untuk menghapus pengumuman yang sudah ada. Untuk menggunakan perintah ini, Anda perlu:

  1. Ketik /announcement-delete [id pengumuman] untuk menghapus pengumuman.
  2. Pastikan Anda memiliki izin yang diperlukan untuk menggunakan perintah ini.
